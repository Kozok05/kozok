# 코족(Kozok) - 큐플레이 아카이브 족보 프로그램 다운로드
큐플레이 아카이브 족보 / 큐플레이 족보 / 오프라인 족보 / 자동 족보

**코족**은 큐플레이 아카이브 서바이벌 게임의 **족보 정답 검색기**입니다.  
검색어를 입력하면 정답을 자동으로 찾아 복사까지 해주는 **무료 족보 프로그램**입니다.

## 주요 기능
- 실시간 서버 한국 표준 시간 표시
- 족보 자동 검색 및 정답 복사
- '꽁꽁', '꼬록', '올라', 'OX/XO', '가로세로' 문제 최신화
- Pretendard 폰트 탑재
- 설치 불필요, 단일 실행파일 제공
- 꽁꽁, 가로세로 정답 자동 복사 및 클릭 복사
- 키워드 검색 시 공백무시 기능
- 키워드 하이라이트
